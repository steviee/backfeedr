## Description
Add/remove apps and rotate API keys.

## Requirements
- List all registered apps
- Create new app (name, bundle ID)
- Show API key (once on creation)
- Rotate API key (invalidate old)
- Delete app (with confirmation)

## UI
- Cards per app
- Copy-to-clipboard for keys
- Danger zone for deletion

## Acceptance Criteria
- [ ] App list displays
- [ ] New app creation works
- [ ] Key rotation works
- [ ] Deletion confirms
