# backfeedr agent context

## Project Overview
Self-hosted crash reporting & metrics for iOS indie developers.
Go + SQLite + HTMX. Single container.

## Schema
Issues track MVP implementation tasks.

Fields:
- id: integer
- title: string
- status: open | in-progress | closed | wontfix
- priority: low | medium | high | critical
- labels: string array

## Commands (manual)
Find issues: ls .issues/*.md
cat .issues/XXXX-<title>.md  # view issue
